## Introduction
Kaboom is a game written in rust.

### Building/Developing
Make sure to install sdl2.
```sh
brew install sdl2
brew install sdl2_image
```
run with:
```sh
cargo run
```

### Dependencies
Entity Component System:
[specs](https://specs.amethyst.rs/)

Graphics:
[sdl2](https://github.com/Rust-SDL2/rust-sdl2)
